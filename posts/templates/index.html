Twitter API by ritikkanswal

BASE URL: https://twitterapi123.herokuapp.com/

1)API to post a text message

  • https://twitterapi123.herokuapp.com/post?text={your text}&lat={latitude}&lon={longitude}

    Example:

  • https://twitterapi123.herokuapp.com/post?text=Ratnagiri&lat=16.994444&lon=73.300003

 

2)API to get nearby recent posts

  • https://twitterapi123.herokuapp.com/nearby?lat={latitude}&lon={longitude}&page{page number}

     Example:

  • https://twitterapi123.herokuapp.com/nearby?lat=28.9845&lon=77.7064&page=1

 

3)API to get current weather

  • https://twitterapi123.herokuapp.com/get?lat={latitude}&lon={longitude}

     Example:

  • https://twitterapi123.herokuapp.com/get?lat=28.9845&lon=77.7064
